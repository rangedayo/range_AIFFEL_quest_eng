# 회고(참고 링크 및 코드 개선)
수업 시간에서 배운 내용 외로 어떤 문장을 표현하고 싶을 때 쓰기에 유용한 메서드를 사용한 부분도 알게 되어서 좋았습니다.
전체적인 구조가 어떻게 되는지 먼저 얘기해주시고, 한줄 한줄씩 왜 이런 코드를 작성했는지 설명해주셔서 코드의 흐름을 이해하기가 수월했습니다.
문제를 보고나서 처음 시작할 때 input 남발만 했었는데, 수식 자체를 입력해도 바로 계산을 할 수 있도록 업그레이드해서 구현하신 것을 보고 대단하는 생각이 들었습니다.
